# MuscleUp Gym Website

A responsive gym website built using **HTML, CSS, and JavaScript**.
This project demonstrates a modern small-business website that can be used by gyms, fitness centers, or personal trainers.

---

## 🚀 Live Demo

(After deployment)

https://muscleup-gym-website.netlify.app

---

## 📌 Features

* Responsive design for desktop and mobile
* Navigation menu
* Hero landing section
* Gym services section
* Trainer profiles
* Membership pricing plans
* Image gallery
* Contact form
* Google Maps location integration

---

## 🛠 Tech Stack
